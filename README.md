## ✨ Features

- **Multiple ML Models**: Random Forest, XGBoost, CatBoost, and Ensemble
- **Interactive UI**: Beautiful Streamlit dashboard
- **Feature Engineering**: Automatic creation of derived features
- **Batch Predictions**: Predict multiple houses at once
- **Confidence Intervals**: Price ranges with 95% confidence
- **Visual Analytics**: Interactive charts and graphs
